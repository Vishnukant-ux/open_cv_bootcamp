# Basic Image Manipulations

Topics Covered:
- Accessing image pixels
- Modifying pixel values
- Cropping images
- Resizing images
- Maintaining aspect ratio during resizing

Key Learnings:
- Images are stored as NumPy arrays.
- Pixels can be accessed using row and column indices.
- Cropping is done using array slicing.
- OpenCV resize() can scale images or set custom dimensions.
- Maintaining aspect ratio prevents image distortion.
