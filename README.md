
## About

This repository serves as a collection of projects and implementations completed as part of the OpenCV Bootcamp.

Throughout this bootcamp, I explored fundamental and advanced computer vision concepts using Python, OpenCV, Deep Learning, and AI-based vision models. Each module is implemented as a separate project repository with source code, documentation, and outputs.
