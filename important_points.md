# HDR Imaging using OpenCV

This demonstrates High Dynamic Range (HDR) imaging using OpenCV.

## Steps

1. Capture multiple exposures
2. Align images
3. Estimate Camera Response Function
4. Merge exposures into HDR image
5. Apply tone mapping

## Techniques Used

- AlignMTB
- Debevec Calibration
- MergeDebevec
- Drago Tonemapping
- Reinhard Tonemapping
- Mantiuk Tonemapping
