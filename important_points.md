# Human Pose Estimation using OpenPose and OpenCV

This project demonstrates single-person human pose estimation using OpenPose and OpenCV DNN.

## Features

- Human keypoint detection
- Skeleton generation
- OpenCV DNN module
- Caffe model inference
- Real-time pose visualization

## Model

- OpenPose MPI Model
- 15 Body Keypoints

## Output

The model detects:

- Head
- Neck
- Shoulders
- Elbows
- Wrists
- Hips
- Knees
- Ankles

and connects them to generate a full skeleton.

## Technologies

- Python
- OpenCV
- Deep Learning
- OpenPose
- Caffe
