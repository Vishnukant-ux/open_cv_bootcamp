### Technologies and Tools Used

This project was developed using Python and OpenCV within Visual Studio Code (VS Code)**. The implementation leverages OpenCV's Deep Neural Network (DNN) module for computer vision tasks such as object detection and human pose estimation.

#### Programming Language

* Python 3.x

#### Libraries Used

* OpenCV
* NumPy
* Matplotlib

#### Development Environment

* Visual Studio Code (VS Code)
* Python Virtual Environment (venv)
* Git & GitHub for version control

#### VS Code Features Utilized

* Integrated Terminal for package installation and execution
* Python Extension for code execution and debugging
* IntelliSense for code completion and suggestions
* File Explorer for project organization
* Git Integration for source control management
* Syntax Highlighting and Error Detection
* Debug Console for troubleshooting and testing

#### Computer Vision Features Implemented

* Image Loading and Processing
* Object Detection using Deep Learning Models
* Human Pose Estimation using OpenPose
* Keypoint Detection
* Skeleton Visualization
* Bounding Box Generation
* Model Inference using OpenCV DNN
* Output Image Generation and Visualization

#### Learning Outcomes

Through this project, I gained practical experience in computer vision workflows, deep learning model deployment, image preprocessing, feature extraction, object detection, and pose estimation using industry-standard tools and frameworks.
